## CLONE REPOSITORY AND ACCESS FILE

* git clone https://github.com/Rodulforo8/booking-app.git

* cd .\booking-app\

## DEPENDENCIES INSTALLATION

* npm install

## RUN SERVER

Run the React js server with the following command on the terminal:

* npm start
